图表与文字配合（inline embed）

这个页面演示：在同一个 HTML 中，用 div 容器包裹 iframe，并用脚本把原项目页面裁切到“只显示目标图表”。这样你可以在任意段落之间插入图表。

例 1：气候诉讼案件分布（地图）

在空间维度上，气候诉讼案件呈现出明显的地域不均衡。下面的地图仅展示目标地图区域，其它 UI 会被隐藏，方便你在论文正文中穿插解释。

图：全球气候诉讼案件分布地图

例 2：森林碳汇潜力矩阵

矩阵图通常用于展示多维度组合（例如变化率与存量）。你可以在这里补充解释象限含义、阈值选择依据等。

图：森林碳汇潜力矩阵

例 3：3D 地球（React/Vite build 产物）

对于 React/Vite 的 3D 地球，我已支持 ?view=globe / ?view=radar，因此可以像普通静态页面一样嵌入。
