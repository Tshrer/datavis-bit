# 🌍 全球气候诉讼可视化分析报告

> Climate Litigation Accountability Analysis

## 📋 项目概述

本项目通过数据可视化技术，分析全球气候变化诉讼的案件分布、被追责主体及诉讼结果，旨在揭示气候问责体系的现状与问题。

**数据来源**：[Sabin Center for Climate Change Law](https://climatecasechart.com/) + [UNEP Global Climate Litigation Report 2025](https://www.unep.org/) + [Grantham Research Institute (LSE)](https://www.lse.ac.uk/granthaminstitute)

---

## 📊 可视化内容

### 1. 全球案件分布地图
- 展示全球40+个国家/地区的诉讼案件数量
- 气泡大小反映案件数量，颜色深浅表示密度
- 支持鼠标悬停查看各国详细数据

### 2. 增长趋势图（1986-2025）
- 展示40年来气候诉讼的增长轨迹
- 支持时间范围刷选，便于聚焦分析特定时期
- 呈现"爆发式"增长特征

### 3. 分类分析图
三种视图切换：
- **起诉方构成**：个人/社区、非政府组织、青年/学生、地方政府的比例
- **被告构成**：国家政府、地方政府、化石燃料公司、其他企业的比例
- **案件结果**：原告胜诉、部分胜诉、被告胜诉、和解、审理中的分布

---

## 🔍 关键发现

### 问题一：地域分布极不均衡

| 地区 | 案件数量 | 占比 |
|------|---------|------|
| 北美（主要为美国） | 5,441 | 59.3% |
| 欧洲 | 1,487 | 16.2% |
| 大洋洲 | 753 | 8.2% |
| 南美 | 423 | 4.6% |
| 亚洲 | 261 | 2.8% |
| 非洲 | 111 | 1.2% |

**问题**：发展中国家（全球南方）案件仅占约9%，与其面临的气候脆弱性严重不匹配。

### 问题二：问责对象偏离核心排放者

| 被告类型 | 案件数量 | 占比 |
|----------|---------|------|
| 国家政府 | 4,680 | 51% |
| 地方政府 | 1,744 | 19% |
| 化石燃料公司 | 1,285 | 14% |
| 其他企业 | 917 | 10% |
| 金融机构 | 550 | 6% |

**问题**：直接起诉化石燃料公司的案件仅占14%，而这些企业是主要排放来源。责任可能被"转嫁"给政府。

### 问题三：原告胜诉率偏低

| 案件结果 | 数量 | 占比 |
|----------|------|------|
| 原告胜诉 | 2,478 | 27% |
| 原告部分胜诉 | 825 | 9% |
| 被告胜诉 | 2,753 | 30% |
| 和解/已解决 | 1,468 | 16% |
| 审理中 | 1,653 | 18% |

**问题**：原告胜诉率（含部分胜诉）约36%，整体偏低，反映现有法律框架对气候侵权认定存在障碍。

### 问题四：时间维度的爆发式增长

- **1986-2000年**：缓慢起步阶段（1→28起）
- **2000-2015年**：稳步增长阶段（28→730起）
- **2015年后**：爆发式增长（730→9,177起）

**背景因素**：
1. 2015年《巴黎协定》签署提升公众气候意识
2. 青年气候运动兴起（如"星期五为未来"）
3. 极端气候事件频发激发诉讼动力
4. 法律框架逐步完善

---

## 💡 核心结论

### 结论一：气候诉讼已成为推动气候行动的重要法律工具

40年间增长超过3,000倍，覆盖40+个司法管辖区，表明：
- 公众气候权利意识显著提升
- 司法途径正在成为政策倡导的重要补充
- 诉讼可能倒逼政府和企业的减排行动

### 结论二：问责体系亟需重新校准

当前问责重心偏向政府而非化石燃料行业，建议：
- 加强对石油、天然气、煤炭公司的直接追责
- 探索"漂绿"（greenwashing）诉讼
- 引入企业气候信息披露强制要求

### 结论三：全球南方需要更多支持

发展中国家诉讼能力与气候脆弱性倒挂，需要：
- 建立国际气候诉讼援助基金
- 加强环境法领域的南南合作
- 支持基层社区的法律赋能

### 结论四：法律改革是关键突破口

2024年欧洲人权法院的里程碑裁定具有历史意义，表明：
- 国际人权法可为气候诉讼提供新依据
- 司法独立性是推进气候行动的重要保障
- 需要建立更完善的气候损害归因科学标准
