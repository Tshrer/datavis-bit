# 🌍 绿色低碳专利全球分布可视化

> 追踪绿色低碳技术的全球分布与技术转让路径

## 📋 项目概述

本项目通过数据可视化技术，直观展示全球绿色低碳专利的区域分布、各国技术实力对比以及国际技术转让流动情况。数据来源于国家知识产权局《绿色低碳专利统计分析报告（2024）》。

---

## ❓ 研究问题

### 核心问题
**在全球应对气候变化的背景下，谁掌握了绿色低碳核心技术的命脉？**

### 具体问题
1. **区域分布不均问题**
   - 全球绿色低碳专利主要集中在哪些国家和地区？
   - 发展中国家与发达国家之间是否存在技术鸿沟？

2. **技术领域失衡问题**
   - 五大技术领域（储能、清洁能源、节能、化石能源降碳、CCUS）的发展是否均衡？
   - 储能领域为何占比最大（37.2%）？

3. **技术转让障碍问题**
   - 绿色低碳技术从发达国家向发展中国家的转让是否顺畅？
   - 中国作为技术大国，对全球技术扩散的贡献有多大？

## 🔍 主要发现与结论

### 1. 中国领跑全球，但差距依然显著

**问题发现：**
- 截至2023年底，中国绿色低碳专利有效量达**24.3万件**，远超其他国家
- 美国（4.2万件）、日本（3.8万件）、韩国（2.8万件）分列其后

**结论：**
> 中国已成为全球绿色低碳技术创新的核心驱动力，对全球增长贡献度达**75.7%**。然而，发展中国家整体技术储备仍与发达国家存在较大差距，技术鸿沟亟待弥合。

### 2. 技术领域发展不均衡

**问题发现：**
| 技术领域 | 占比 | 增速 |
|----------|------|------|
| 储能 | 37.2% | +19.8% |
| 清洁能源 | 25.0% | +15.2% |
| 节能与能量回收 | 20.0% | +12.8% |
| 化石能源降碳 | 11.0% | +10.5% |
| CCUS | 6.7% | +8.3% |

**结论：**
> 储能技术成为最热门领域，反映出全球对新能源存储的迫切需求。相比之下，碳捕集利用封存（CCUS）技术发展相对滞后，仅占6.7%，需加强研发投入。

### 3. 技术转让存在不对称性

**问题发现：**
- 技术流向主要从中国、美国、日本向印度、巴西、越南等发展中国家流动
- 中国对发展中国家的技术转让最为活跃

**结论：**
> 国际技术转让呈现"东方崛起"态势，中国正成为南南技术合作的重要桥梁。但发达国家对核心技术的保护意识依然较强，全面技术共享仍面临挑战。

### 4. 区域集聚效应明显

**问题发现：**
- 亚洲国家（中国、日本、韩国）占据全球绿色低碳专利主导地位
- 欧洲（德国、法国、英国）与北美（美国、加拿大）形成第二梯队
- 非洲、南美国家专利数量相对较少

**结论：**
> 技术分布呈现明显的"亚洲+北美/欧洲"双核心格局。发展中国家需加强区域合作，构建适合本地需求的绿色技术体系。

## 🎯 政策建议

### 对发展中国家
1. **加强技术引进**：充分利用国际技术转让渠道，特别是与中国等新兴技术大国的合作
2. **聚焦储能领域**：优先发展新能源存储技术，把握能源转型关键节点
3. **培育创新生态**：完善知识产权保护制度，激发本土创新活力

### 对发达国家
1. **开放技术共享**：在应对气候变化的共同目标下，减少技术壁垒
2. **支持能力建设**：帮助发展中国家提升绿色技术研发与应用能力

### 对国际社会
1. **建立技术公平机制**：确保发展中国家能够以合理成本获取绿色低碳技术
2. **加强监测评估**：建立全球绿色技术专利动态监测体系
